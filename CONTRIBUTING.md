# 参与反馈 / Contributing Feedback

## 中文

当前仓库主要用于 LinkGo 的产品介绍、发布包和用户反馈，不放完整源码。

欢迎反馈：

- 文件格式处理问题。
- 翻译或校对质量问题。
- 模型配置问题。
- LinkClaw 本机沙箱问题。
- 安装包和运行环境问题。
- 用户界面和交互建议。

反馈时建议提供：

- 操作系统和版本。
- 文件类型和大致结构。
- 你输入的任务描述。
- 预期结果和实际结果。
- 脱敏后的日志或截图。

请不要上传真实客户文件、API Key、Token、Cookie、数据库或其它隐私信息。

## English

This repository is used for LinkGo product information, release packages, and user feedback. It does not host the full source code.

Feedback is welcome for:

- File format handling issues.
- Translation or review quality issues.
- Model configuration issues.
- LinkClaw local sandbox issues.
- Installer and runtime environment issues.
- UI and interaction suggestions.

When reporting, please include:

- Operating system and version.
- File type and rough structure.
- The task instruction you entered.
- Expected result and actual result.
- Sanitized logs or screenshots.

Please do not upload real customer files, API keys, tokens, cookies, databases, or private information.
