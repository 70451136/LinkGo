# 功能总览

## 文件能力

- Word：翻译、双语对照、译文写回、表格提取、译文校对。
- Excel：翻译、样式保留、表格整理、数据导出。
- PowerPoint：翻译、文本框边界守护、版式检查。
- PDF：解析、翻译、校对、结果导出。
- 字幕：SRT 等字幕翻译、时间轴保留、双语字幕。
- 图片：OCR 识别和翻译。
- 视频：字幕提取、翻译和交付。
- EPUB / TXT / Markdown / SDLXLIFF / DXF 等更多格式。

## 智能体能力

- 文件翻译智能体。
- 译文校对智能体。
- Office 排版守护。
- LinkClaw 本机沙箱。
- 联网检索与来源整理。
- 术语库、翻译记忆和长期记忆召回。

## 模型能力

- 在线模型：OpenAI、OpenRouter、DeepSeek、火山方舟、阿里云、智谱、月之暗面、百度、硅基流动等。
- 本地模型：Ollama、LM Studio、vLLM 等。
- 支持按任务选择模型，也支持设置默认模型。

## 管理能力

- 用户和权限管理。
- 模型配置。
- 术语库和记忆中心。
- 灵感库内容管理。
- 本地运行时和打包环境适配。
