# 功能总览 / Feature Overview

## 中文

### 文件处理能力

| 类型 | 能力 |
| --- | --- |
| Word | 翻译、双语对照、译文写回、表格提取、译文校对、格式守护 |
| Excel | 翻译、样式保留、表格整理、批注/富文本处理、数据导出 |
| PowerPoint | 翻译、文本框边界守护、版式检查、演示稿内容整理 |
| PDF | 解析、翻译、校对、结果导出、扫描件 OCR 辅助 |
| 字幕 | SRT 等字幕翻译、时间轴保留、双语字幕 |
| 图片 | OCR 识别、图片文字翻译、内容说明 |
| 视频 | 字幕提取、字幕翻译、视频字幕交付 |
| 其它 | EPUB、TXT、Markdown、SDLXLIFF、DXF/DWG 等更多格式 |

### 翻译与校对能力

- 保留格式翻译。
- 双语对照输出。
- 翻译记忆复用。
- 术语库约束。
- 长文档分片处理。
- 漏译、错译、术语、数字单位和语法检查。
- 校对结果表格化输出。

### 智能体能力

- 文件翻译智能体。
- 译文校对智能体。
- Office 排版守护。
- LinkClaw 本机沙箱。
- 联网检索与来源整理。
- 多步骤任务续接。
- 本机目录和文件处理。

### 模型能力

- 在线模型：OpenAI、OpenRouter、DeepSeek、火山方舟、阿里云、智谱、月之暗面、百度、硅基流动等。
- 本地模型：Ollama、LM Studio、vLLM 等。
- 支持不同任务选择不同模型。
- 支持智能体、文件翻译和普通聊天分别配置模型。

### 记忆与知识能力

- 术语库。
- 翻译记忆库。
- 长期记忆。
- 用户偏好。
- 项目风格沉淀。
- 附件语义召回。

### 管理能力

- 用户和权限管理。
- 模型配置。
- 本地运行时部署检测。
- 术语库和记忆中心。
- 灵感库内容管理。
- 发布页和静态内容管理。

## English

### File Processing

| Type | Capabilities |
| --- | --- |
| Word | Translation, bilingual output, writeback, table extraction, review, layout preservation |
| Excel | Translation, style preservation, table cleanup, comments/rich text handling, data export |
| PowerPoint | Translation, text-box boundary protection, layout checks, deck content organization |
| PDF | Parsing, translation, review, export, OCR-assisted scanned document handling |
| Subtitles | SRT translation, timing preservation, bilingual subtitles |
| Images | OCR, image text translation, content explanation |
| Video | Subtitle extraction, subtitle translation, video subtitle delivery |
| Others | EPUB, TXT, Markdown, SDLXLIFF, DXF/DWG, and more |

### Translation and Review

- Format-preserving translation.
- Bilingual output.
- Translation memory reuse.
- Glossary enforcement.
- Long-document chunking.
- Missing translation, mistranslation, terminology, number/unit, and grammar checks.
- Structured review tables.

### Agent Capabilities

- File translation agent.
- Translation review agent.
- Office layout guard.
- LinkClaw local sandbox.
- Web research and source organization.
- Multi-step task continuation.
- Local directory and file processing.

### Model Capabilities

- Online providers: OpenAI, OpenRouter, DeepSeek, Volcano Ark, Alibaba Cloud, Zhipu AI, Moonshot, Baidu, SiliconFlow, and more.
- Local providers: Ollama, LM Studio, vLLM, and more.
- Per-task model selection.
- Separate model settings for chat, file translation, and agents.

### Memory and Knowledge

- Glossaries.
- Translation memories.
- Long-term memory.
- User preferences.
- Project style memory.
- Semantic recall for attachments.

### Administration

- User and permission management.
- Model configuration.
- Local runtime detection and deployment.
- Terminology and memory center.
- Content library management.
- Product pages and static content management.
