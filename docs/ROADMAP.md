# 路线图

## 近期重点

- 完善发布包下载和安装体验。
- 持续增强 Office / PDF / 字幕的保留格式翻译。
- 优化 LinkClaw 与 OpenClaw / Hermes 的实时输出和文件工具链。
- 完善在线模型和本地模型的自动适配。
- 强化译文校对表格输出和术语一致性检查。

## 中期方向

- 更稳定的批量文件夹处理。
- 更完整的企业术语和翻译记忆协作。
- 更清晰的本机沙箱权限提示和审计日志。
- 更多行业模板：法律、医学、制造、跨境电商、软件本地化。

## 长期方向

- 让 LinkGo 成为本机优先的 AI 文件工作台。
- 让用户用自然语言完成从文件输入到可交付结果的完整流程。
