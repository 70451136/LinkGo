# 路线图 / Roadmap

## 中文

### 近期重点

- 完善 Windows / Linux 发布包。
- 优化安装、启动和本地运行时自动部署体验。
- 增强 Word、Excel、PPT、PDF 和字幕的保留格式翻译。
- 优化译文校对表格输出，让问题更明确、更可修改。
- 改进 LinkClaw 与 OpenClaw / Hermes 的实时过程输出。
- 完善在线模型和本地模型的统一选择与自动适配。

### 中期方向

- 更稳定的文件夹批量处理。
- 更完整的企业术语库和翻译记忆协作。
- 更多行业模板：法律、医学、制造、跨境电商、软件本地化。
- 更细致的本机沙箱权限提示和审计日志。
- 更丰富的灵感库、产品案例和用户教程。

### 长期方向

- 成为本机优先的 AI 文件工作台。
- 让用户用自然语言完成从文件输入到可交付结果的完整流程。
- 让模型、工具、记忆和智能体协作得更自然、更可靠。

## English

### Near-Term Focus

- Improve Windows / Linux release packages.
- Improve installation, startup, and local runtime bootstrap experience.
- Strengthen format-preserving translation for Word, Excel, PowerPoint, PDF, and subtitles.
- Improve translation review tables with clearer issues and actionable suggestions.
- Improve real-time progress output for LinkClaw with OpenClaw / Hermes.
- Unify model selection and adaptation across online and local models.

### Mid-Term Direction

- More reliable folder-level batch processing.
- Better enterprise glossary and translation memory collaboration.
- More domain templates: legal, medical, manufacturing, cross-border commerce, and software localization.
- More granular local sandbox permission prompts and audit logs.
- Richer product examples, user guides, and content library material.

### Long-Term Vision

- Become a local-first AI workspace for real file workflows.
- Let users go from file input to deliverable output through natural language.
- Make models, tools, memories, and agents collaborate more naturally and reliably.
