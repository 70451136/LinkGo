# 发布包说明 / Release Packages

## 中文

本仓库后续会通过 GitHub Releases 发布 LinkGo 安装包、便携包和更新说明。

### 计划发布内容

- Windows 桌面运行包。
- Linux / Ubuntu 部署包。
- 更新说明。
- 已知问题。
- 校验信息。

### 安装包可能包含或自动检测

- Python 运行时。
- Node.js。
- Git Bash。
- WSL / Linux 兼容环境。
- Redis、MySQL、Qdrant 等本地服务。
- Ollama / vLLM 等本地模型运行时。
- LinkClaw 所需的 OpenClaw / Hermes 适配环境。

### 升级前建议备份

- 用户数据库。
- 术语库和翻译记忆。
- 模型配置。
- 本机沙箱输出目录。
- 自定义提示词和长期记忆。

### 下载位置

请关注 [GitHub Releases](https://github.com/70451136/LinkGo/releases)。

## English

Future LinkGo installers, portable packages, and release notes will be published through GitHub Releases.

### Planned Release Assets

- Windows desktop package.
- Linux / Ubuntu deployment package.
- Release notes.
- Known issues.
- Checksums.

### Packages May Include or Detect

- Python runtime.
- Node.js.
- Git Bash.
- WSL / Linux compatibility environment.
- Local services such as Redis, MySQL, and Qdrant.
- Local model runtimes such as Ollama / vLLM.
- LinkClaw runtime requirements for OpenClaw / Hermes integration.

### Before Upgrading

Consider backing up:

- User database.
- Glossaries and translation memories.
- Model configuration.
- Local sandbox output directories.
- Custom prompts and long-term memories.

### Download Location

Please watch [GitHub Releases](https://github.com/70451136/LinkGo/releases).
