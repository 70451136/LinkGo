# LinkClaw 本机智能体

LinkClaw 是 LinkGo 的本机智能体桥接模块，用于接入 OpenClaw 和 Hermes Agent。

## 它解决什么问题

普通聊天适合回答问题；LinkClaw 更适合让智能体在本机工作区里完成任务，例如读取目录、调用文件管道、运行命令、生成文件和处理多步骤任务。

## 关键特点

- 可选择 OpenClaw 或 Hermes 作为执行引擎。
- 支持只读、标准、完整权限模式。
- 同步主项目模型、自定义提示词、术语库、记忆库、温度和上下文设置。
- 本机沙箱只允许本机访问，避免远程误操作电脑。

## 典型任务

- 检查目录里的原文和译文 Word，输出校对表格。
- 把上传的 PPT 翻译成韩语并保留排版。
- 生成一个 GUI 小工具或脚本。
- 批量整理工作区中的临时文件。

## 安全边界

完整权限会允许更强的本机操作能力。使用前请确认任务目标明确，并避免把沙箱目录设置为系统关键目录。
