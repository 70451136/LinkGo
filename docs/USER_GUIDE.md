# 用户指南

## 1. 基本使用方式

1. 打开 LinkGo。
2. 选择在线模型或本地模型。
3. 上传文件或选择本机沙箱目录。
4. 用自然语言描述目标。
5. 查看回答、下载结果文件，必要时继续追问。

## 2. 文件翻译

你可以直接说：

- 把这个文件翻译成日语，保留格式和排版。
- 这个 PPT 翻译成英文，不要超出文本框。
- 把这个字幕翻译成韩语，保留时间轴。
- 翻译这个 Word，并输出中英对照。

LinkGo 会尽量复用原格式写回管道，避免把 Office 文件简单转换成纯文本。

## 3. 译文校对

你可以上传：

- 一个双语对照文件。
- 一个原文文件和一个译文文件。
- 表格形式的原译文对照材料。

然后输入：

- 检查这个文件翻译得怎么样。
- 帮我校对，输出表格结果。
- 检查数字、术语、漏译和错译。

## 4. 本机沙箱

本机沙箱用于处理电脑上的本地目录和文件。它只应在运行 LinkGo 的电脑上使用。

权限模式建议：

- 只读权限：适合分析文件、总结内容。
- 标准权限：适合生成结果文件。
- 完整权限：适合明确需要执行命令、操作本机或批量处理文件的任务。

## 5. 模型选择

普通聊天和简单摘要可以选择更快的模型。复杂文件校对、长文档处理、代码和本机自动化建议选择能力更强的模型。

如果你更关注隐私和本机优先，可以使用 Ollama、LM Studio 或 vLLM 等本地模型。
