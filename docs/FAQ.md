# 常见问题

## LinkGo 是开源项目吗？

当前 GitHub 仓库主要作为产品介绍和发布包入口，不直接放完整源码。后续发布包会通过 Releases 提供。

## 什么时候可以下载？

打包版本准备好后会上传到 Releases。你可以 Watch 本仓库关注更新。

## 支持哪些文件格式？

重点支持 Word、Excel、PPT、PDF、字幕、图片 OCR、视频字幕、EPUB、TXT、Markdown、SDLXLIFF、DXF 等格式。具体能力会随版本持续更新。

## 能完全保留格式吗？

LinkGo 会尽量保留格式和排版，但复杂文件可能仍需要人工复核。尤其是 PDF、复杂 PPT、嵌套表格、扫描件和特殊字体文件。

## 可以使用本地模型吗？

可以。LinkGo 目标支持 Ollama、LM Studio、vLLM 等本地模型，也支持多家在线模型。

## LinkClaw 是什么？

LinkClaw 是本机智能体桥接模块，用于接入 OpenClaw / Hermes，让它们在 LinkGo 的本机沙箱和文件管道中工作。

## 我该如何反馈问题？

提交 Issue，尽量包含操作系统、文件类型、任务目标、预期结果、实际结果和必要的脱敏日志。
