# LinkGo

**本机智能体与多格式 AI 翻译工作台**

LinkGo 面向译员、企业团队、内容创作者和需要本机自动化的高级用户。它把多格式文件翻译、原格式写回、术语库、翻译记忆、本地模型、在线模型和本机智能体整合到一个桌面级 Web 工作台里。

> 上传文件，说出目标，LinkGo 会尽量保留原始格式和排版，自动选择合适的模型、工具和智能体，把结果整理成可查看、可下载、可继续追问的交付物。

## 你可以用 LinkGo 做什么

- 把 Word、Excel、PPT、PDF、EPUB、字幕、图片 OCR 或视频字幕翻译成目标语言。
- 尽量保留原文格式、字体、表格、文本框、版式和字幕时间轴。
- 检查双语译文质量，输出漏译、错译、术语、数字单位和表达问题。
- 从文档中提取表格，整理为 Excel 或结构化结果。
- 通过 LinkClaw 本机沙箱连接 OpenClaw / Hermes，让智能体处理本机目录、文件和自动化任务。
- 使用术语库、翻译记忆和长期记忆，让后续工作更贴合你的行业和表达习惯。

## 核心亮点

### 多格式保留排版翻译

LinkGo 关注真实办公文件，而不是只翻译纯文本。它会尽量保持 Office 文档、字幕、PDF 和其它文件的结构，让译文更接近可交付状态。

### 译文校对与质量检查

支持单文件双语对照，也支持原文文件 + 译文文件。适合检查合同、技术文档、医学材料、产品资料、商务通知和多领域测试稿。

### 术语库与翻译记忆

项目、团队和用户可以逐步积累术语、译法、风格和长期偏好。翻译、校对和智能体任务可以共享这些上下文。

### LinkClaw 本机智能体

LinkGo 可通过 LinkClaw 接入 OpenClaw / Hermes Agent，把主项目的模型、提示词、术语库、记忆库、温度、上下文和权限设置传递给本机智能体。

### 在线模型与本地模型

支持 OpenAI、OpenRouter、DeepSeek、火山方舟、阿里云、智谱、月之暗面、百度、硅基流动等在线模型，也支持 Ollama、LM Studio、vLLM 等本地模型。

## 使用场景

| 场景 | 用户怎么说 | LinkGo 输出 |
| --- | --- | --- |
| 文件翻译 | 把这个 Word 翻译成日语，保留格式 | 可下载译文文件 |
| 译文校对 | 检查这个双语文件翻译得怎么样 | 校对报告和问题表格 |
| 表格提取 | 把 Word 里的表格转成 Excel | XLSX 文件 |
| 字幕处理 | 翻译这个视频字幕并保留时间轴 | 字幕文件或视频字幕结果 |
| 本机沙箱 | 清理这个工作区文件 / 生成一个小工具 | 本机智能体执行结果 |
| 资料整理 | 总结这批文件，列出重点 | 结构化摘要 |
